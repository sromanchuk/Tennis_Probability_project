data_processing module¶

data_processing.get_json(source)¶

(str) -> dict Get json data from api.sportradar.us with given link. :param source: link :return: dictionary

data_processing.matches_from_tournaments(file)¶

(str) -> list Get matches from tournaments file. :param file: path :return: list of match ids

data_processing.player_profile(id, rankings_path)¶

(str, str) -> list Get information about player. :param id: player id :param rankings_path: path of rankings file :return: list

data_processing.previous_encounters(id1, id2, date)¶

(str, str, datetime.datetime) -> float Get performance statistics during previous encounters of 2 player before given date. :param id1: first player :param id2: second player :param date: datetime object :return: winrate

data_processing.rank(id, rankings)¶

(str, list) -> int Return rank of player by id. :param id: player id :param rankings: rankings list :return: player rank

data_processing.rankings(gender)¶

(str) -> lst :param gender: 0 for female rankings, 1 for male rankings :return: list containing rankings

data_processing.surface_stats(data)¶

(dict) -> dict Get surface performance statistics from given player data. :param data: player dict :return: dict of surface performances

data_processing.tournaments(gender)¶

(int) -> dict Get tournaments for given gender. :param gender: 0 for females, 1 for males :return: dict
